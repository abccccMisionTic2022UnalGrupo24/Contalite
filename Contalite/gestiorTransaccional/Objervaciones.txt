Para Agregar o modificar trasnacciones el ID del usuario debe existir en el momento que se use el formato Json
{
  "id_tr": 14562,
  "tipo": "Ingreso",
  "fecha_tr": "2022-09-09",
  "usuario_id": 1,
  "concepto": "Ingreso salarios",
  "monto": 200000
}

Para agregar empresas no se requiere dato historico( es neceasrio realizar caractrizacion para solo usuarios admin)

Para agregar empleados es necesario que exista el id de empresa y un id de usuario  el cual esta relacionado con el nit en empresas
